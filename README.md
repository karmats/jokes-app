# Angular codegames

Project for Wolters Kluwer codegames.

## Step 1 - Initialize project

```bash
$ ng new jokes-app
```

- Stricter type checking -> No
- Routing -> No
- Stylesheet format -> CSS

Verify that it works with

```
ng serve
```

## Step 2 - Scaffolding

- app.component.html

```html
<h1>Hello world!</h1>
```

- styles.css
